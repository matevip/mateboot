## 平台说明

MateBoot是一个基于SpringBoot3的快速开发平台，采用前后端分离的模式，前端采用Element Plus组件，后端采用SpringBoot3、Sa-token、Mybatis-Plus、Redis、RabbitMQ、FastDFS等核心技术，同时提供了代码生成器，能够快速生成前后端代码，减少开发量，提高开发效率。
